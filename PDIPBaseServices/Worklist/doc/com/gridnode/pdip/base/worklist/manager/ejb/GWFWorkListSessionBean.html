Class GWFWorkListSessionBean

public class GWFWorkListSessionBean
implements SessionBean
GWFWorkListSessionBean is a WorkList Data Access Controller Bean. It is essentially responsible to act as WorkListManager, which performs the basic services such as, 1. Insert the WorkItems. 2. Reject WorkItems. 3. Assign WorkItems. The core business logic for tasks's such as assignWorkItems() - are subject to change respective to the version.
