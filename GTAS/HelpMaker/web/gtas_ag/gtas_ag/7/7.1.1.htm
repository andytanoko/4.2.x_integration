7.1.1 To create your own channel

Step 1. Click Communication. Take note that the All Channels list view page is displayed by default.

Step 2. Click Own under Channels to view your own channels defined earlier.

Step 3. In the Own Channels page, click Create Channel Info. Then the screen appears.

Step 4. Enter the following fields for the Channel Info setting:

Name:

This field gives the name for the channel. Enter Name.

Description:

This field gives the descriptive purpose for that channel. Enter Description.

Protocol Type:

This field allows you to select the protocol type for that channel. The available protocols are HTTP, JMS, SOAP-HTTP and UPC. SOAP-HTTP is a protocol used for transporting SOAP messages over HTTP.

Is Partner:

This field determines if the channel is a partner. If the channel belongs to the owner, just leave the checkbox unchecked.

Communication Profile:

This field provides the communication details for that channel. The details are based on the host, port and protocol type of that channel.

Click on the drop-down arrow next to this field to select your entry. If you don’t find any available selection, click button to create a new communication profile.

Packaging Profile:

This field provides the packaging details for that channel. The details are based on the envelope of that packaging profile. An envelope is a specification that encapsulates or precedes document data for transport, message packaging and security. There are two types of envelope in GridTalk to support RosettaNet messages. They are RNIF1 and RNIF2.

RNIF1 specifies that responses must always follow signals. This means that a workflow might receive a response after the receipt acknowledgement. RNIF2 allows out-of-sequence receipt of signals. For example, a workflow might receive a response before the receipt acknowledgment.

There is a third type of envelope, which is AS2. This means that the document will be transmitted as using the AS2 standard format (RFC 4130).

Click on the drop-down arrow next to this field to select your entry. If you don’t find any available selection, click button to create a new packaging profile.

Security Profile:

This field provides the security details for that channel. The details are based on the type of encryption. There are two types of encryption such as S/MIME and Asymmetric (Public Key Cryptography). Click on the drop-down arrow next to this field to select your entry. If you don’t find any available selection, click button to create a new security profile.

Step 5. Click Save Channel Info. The new channel will be displayed in the Channel Info list view page.

Step 6. To edit your own channel, click on the Edit button next to the channel you have just selected. Update the entries of the channel as necessary. Click Save Channel Info.

Step 7. To delete your own channel, select the check box of the channel you want to delete. Click Delete Channel Info. The selected channel will be removed from the list.
