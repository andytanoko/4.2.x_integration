7.3.1 To create your own security profile

Step 1. Click Communication.

Step 2. Click Own under Security Profiles.

Step 3. In the Own Security Profiles list view page, click Create Security Profile. The screen appears.

Step 4. Insert the necessary fields. Some fields that are highlighted blue are mandatory.

The mentioned fields are described:

Name:

This field gives the name for the security profile.

Description:

This field gives the descriptive purpose for that security profile.

Partner:

This field specifies if that security profile is a partner or not. For own partner security profile, ensure it is unchecked.

Encryption:

This field gives three options to select. The options are None, S/MIME and Asymmetric (Public Key Cryptography). If you choose Asymmetric (Public Key Cryptography), the Encryption Level and Encryption Certificate will be enabled. For the S/MIME encryption chosen, the Encryption Level, Encryption Certificate and Encryption Algorithm will be enabled for you to select.

The Encryption Level determines how difficult your level is set so that an unauthorized user is unable to read the encrypted message. For Asymmetric (Public Key Cryptography), the different levels available are 1024 bits, 512 bits, 256 bits and 64 bits. For S/MIME, the different levels available are 128 bits, 168 bits, 40 bits and 64 bits.

The Encryption Certificate is used for encrypting messages. You should select own GridTalk Certificate or imported Private Certificate.

The Encryption Algorithm is used to indicate the algorithm to use to encrypt messages. Supported algorithms are 3DES and RC2. Using 3DES algorithm requires the use of 168-bit encryption level only.

Step 5. Click Save Security Profile. The newly created own security profile will be displayed in the All Security Profiles and Own Security Profiles list view pages.

Step 6. To edit your own Security Profile, click on the Edit button next to the profile you have just selected. Update the entries as necessary. Click Save Security Profile.

Step 7. To delete your Own Security Profile, select the check box of the own security profile you want to delete. Click Delete Security Profiles. The selected security profile will be removed from the All Security Profiles and Own Security Profiles list view pages.
