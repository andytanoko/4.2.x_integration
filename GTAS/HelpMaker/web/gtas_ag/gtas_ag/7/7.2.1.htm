7.2.1 To create your own communication profile

Step 1. Click Communication.

Step 2. Click Own under Communication Profiles

Step 3. In the Own Communication Profile list view page, click Create Communication Profiles. The screen appears.

Step 4. Insert the necessary fields. Some fields that are highlighted blue are mandatory.

The named fields are described as:

Name:

This field gives the name for the Communication Profile.

Description:

This field gives the descriptive purpose for that Communication Profile.

Partner:

This field specifies if that Communication Profile is a partner or not. For Own communication profile, this is unchecked.

Protocol Type:

This field gives the choices of protocols to configure the comm. setting. The available protocols are HTTP, JMS, SOAP-HTTP and UPC. See more details on Configuring GTAS for HTTPs connection.

Url:

This field specifies the website or server that communicates with GridTalk. For the Protocol type, when that type is selected, the URL format will be displayed to help you write the URL.

Step 5. Click Save Communication Profile. The newly created Own Communication Profile will be displayed in the All Communication Profiles and Own Communication Profile list view pages.

Step 6. To edit your own communication profile, click on the Edit button next to the profile you have just selected. The screen appears. Update the entries as necessary. Click Save Communication Profile.

Step 7. To delete your own communication profile, select the check box of the profile you want to delete. Click Delete Communication Profiles. The selected profile will be removed from the All Communication Profiles page and the Own Communication Profile page.
