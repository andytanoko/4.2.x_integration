7.5.1 To Import Own/Partner Certificate

Step 1. Click on the Communication hyperlink. This will bring you to the Communication left panel where you can select and click Certificates. The Certificates list view page is displayed with the list of imported Partner and Own certificates.

Step 2. Click Import Certificate. The Import Certificate detail view page appears.

Step 3. Proceed to the Partner Certificate field. Select No for the Own Certificate and then you will be prompted to enter the password. As for the Partner Certificate, select Yes and no password will be prompted.

Step 4. For Partner Certificate, you will be prompted for the Is CA field. Select Yes if the certificate is a CA certificate, select No if it is just a normal partner certificate.

Step 5. Click Browse to select the certificate file. The Choose file window is displayed. Browse the list of the certificate files you might have stored in your system. Once you have selected the file, the file will appear under the Certificate File field.

Take note that for Own Certificate, it only supports “.p12” or “.pfx (PKCS#12). For Partner Certificate, it only supports “.cer” (DER encoded format).

Step 6. If you wish to view the details of the Issuer and Subject, you can view the details by clicking on the Add/Remove File(s) button. If the selected certificate file is valid, you will see the details under their respective tabs.

Step 7. Using the same method as above step, you can also view the Serial Number, Valid From and Valid To of the selected certificate.

Step 8. If you wish to replace the existing certificate, just go to the As Pending Certificate For field to select the existing certificate to be replaced when it expires. The new certificate will be automatically used when the old certificate expires. If you select this field, upon saving, the old certificate will have the On expire, use field linked to this new certificate.

Step 9. Click Save Imported Certificate to import the selected certificate into GridTalk.

Step 10. To remove an imported Certificate from GridTalk, select the check box of the Certificate you want to delete. Click Delete Certificate(s). The selected certificate will be removed from the All Certificates and one of the Own Certificates or Partner Certificates list view pages, depending on whether it was a own Certificate or partner Certificate.
