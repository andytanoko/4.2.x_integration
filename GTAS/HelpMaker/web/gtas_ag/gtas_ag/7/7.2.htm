7.2 Configuring Communication Profiles

A Communication Profile is a set of properties that enables communication with partners involved in the business process using the information based on the type of protocol, that is, HTTP, JMS, SOAP-HTTP, and UPC.

You can modify your own communication profile and profiles of partners.

When you click Communication Profiles under the Communication folder, the All Communication Profiles page is displayed by default. You can choose to click Own under Communication Profiles if you want to view your Own Communication Profile page instead of the All Communication Profiles page for easier viewing.
