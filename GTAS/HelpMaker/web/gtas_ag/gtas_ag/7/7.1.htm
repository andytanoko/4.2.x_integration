7.1 Configuring Channels

A channel is a set of properties that designates the source of documents and defines specific processing steps that are performed by GridTalk before a document is delivered to the destination designated by the messaging port or protocol (for example, HTTP, JMS, SOAP-HTTP) with which the channel is associated. The available properties of creating a channel contain the properties of its communication, security and packaging profiles.

You can modify your own channels and that of partners.

When you click Communication tab, the All Channels page is displayed by default. You can choose to click Own under Channels if you want to view the Own Channel page instead of the All Channels page for easier viewing.

A channel is a type of profile used to communicate with another peer. The profile includes the following:

  • Communication data (e.g. protocol type, host, port, etc)
  • Packaging (e.g. type of envelope, etc)
  • Security (e.g. encryption length, type of encryption, etc)
  • Certificate for encryption and decryption
