7.5.2 To download and install the Imported Own/Partner Certificate

Step 1. Click on the Communication hyperlink. This will bring you to the Communication left panel where you can select and click Certificates. The Certificates page is displayed with the list of imported Partner and Own certificates.

Step 2. Select and click the certificate from the list table of imported certificates. The browser brings you to view or edit the certificate. Not only but also you can download the certificate file by clicking it at the Certificate File field.

Step 3. Click it. Then the Download File dialog box appears. You are advised to choose to save it to your directory folder.

Step 4. Once the certificate file is downloaded and saved, click Open. You will see that the Certificate dialog box appears.

Step 5. To view more details of the file, click on the Details tab. The Details page appears.

Step 6. Proceed to click on the Certification Path tab. The Certification Path page appears.

Step 7. Go to the General tab to click Install Certificate. The Certificate Import Wizard appears.

Step 8. Click Next.

Step 9. Click Next.

Step 10. Click Finish. Then the message box will say, “The import was successful”. Continue to click OK.
