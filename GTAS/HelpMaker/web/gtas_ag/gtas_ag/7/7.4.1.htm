7.4.1 To create your own packaging profile

Step 1. Click Communication.

Step 2. Click Own under Packaging Profiles.

Step 3. In the Own Packaging Profiles list view page, click Create Packaging Profile. The screen appears.

Step 4. Insert the necessary fields. Some fields that are highlighted blue are mandatory.

The fields are described as:

Name:

This field gives the name for the packaging profile.

Description:

This field gives the descriptive purpose for that packaging profile.

Partner:

This field specifies if that packaging profile is a partner or not. For own packaging profile, ensure this is unchecked.

Envelope:

This field gives you three options to select. The options are AS2, None, RNIF1 and RNIF2. If the Envelop type is RNIF1 or RNIF2, the message will be processed accordingly to the RNIF packaging (RosettaNet) standards. If you select None, the message will be processed accordingly to GridTalk standard. If you select AS2, the additional settings for AS2 packaging profile extension will be required. Refer to Chapter 16.2 Configuring AS2 Messaging.

Step 5. Click Save Packaging Profile. The newly created own packaging profile will be displayed in the All Packaging Profiles and Own Packaging Profiles list view pages.

Step 6. To edit your Own Packaging Profile, click on the Edit button next to the profile you have just selected. In the Packaging Profile list, update the entries as necessary. Click Save Packaging Profile.

Step 7. To delete your Own Packaging Profile, select the check box of the own packaging profile you want to delete. Click Delete Packaging Profiles. The selected packaging profile will be removed from the All Packaging Profiles and Own Packaging Profiles list view pages.
