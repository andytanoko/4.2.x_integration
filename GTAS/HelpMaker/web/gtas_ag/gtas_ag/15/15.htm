15 Export and Import Configuration

Incorporated with the Import and Export Configuration features, GridTalk is designed to allow the administrator to export configuration data for backup and import the configuration data back to the application when need arises.

All configuration records are exportable (and hence importable) except configurations under the Communication and Registry tabs, User Accounts, Schedule Tasks, and Business Certifcate Mappings.

Take note that system configurations could not be exported or imported if they are not editable. However, you can export or import your user-defined configurations.

When need arises, you may like to revert you configuration records to original settings; this is when you may import the configurations back to the application to reset the configurations back to earlier settings.
