8.5 Managing AS2 Doc Type Mappings

The user usually make use of the AS2 module to transfer EDI document (EDI-X12, EDI-FACT), the capability of identifying the type of the EDI document will further improve the way system handling the incoming document. For example, the system can trigger different partner function for different type of incoming EDI document .

During runtime, the system will determine the type of the EDI document and map to the corresponding business document type. Refer to Chapter 8.1 for more detail.

If the system is not able to determine the type of the EDI document or there is no corresponding AS2 Doc Type Mapping record, the business document type will be set to “AS2 Message”.
