8.2.4 To create a new DTD mapping file

DTD is used to define the legal elements of an XML document.

The purpose of a Document Type Definition is to define the legal building blocks of an XML document. It defines the document structure with a list of legal elements. A DTD can be declared inline in your XML document, or as an external reference.

Why use a DTD? With DTD, your XML files can carry a description of its own format with it. With a DTD, independent groups of people can agree to use a common DTD for interchanging data. Your application can use a standard DTD to verify that the data you receive from the outside world is valid. You can also use a DTD to verify your own data.

DTDs can be created using a specialized DTD editor program or any text-based program. Once created, DTDs can be inserted into GridTalk to allow GridTalk to use these DTDs.

Step 1. Click Mapping.

Step 2. Click DTD under Mapping Files. The DTD Mapping Files list view page is displayed.

Step 3. Click Create Mapping File. The Mapping File detail view page is displayed.

Step 4. In the Mapping File detail view page, enter the mandatory fields.

Step 5. Select DTD for the Type field.

Step 6. Select the file by clicking Browse. The File selector is displayed where you can select a DTD file.

Step 7. Click Save Mapping File. The new mapping file will be displayed in the Mapping Files or DTD Mapping Files list.

Step 8. To add DTD mapping files, repeat the whole procedure. If you want to view the mapping file, go to the DTD Mapping Files page where you will see the records listed in the table. Click on the selected mapping file’s hyperlink under Name in the table. If you wish to update the selected mapping file, click Edit Mapping File.

Step 9. To edit the DTD mapping file, select the mapping file and then click on the corresponding Edit button. The Mapping File page is displayed. Update the entries of the mapping file as necessary. Click Save Mapping File.

Step 10. To delete the DTD mapping file, tick the Select checkbox of the mapping file you want to delete. The mark appears in the checkbox. Click Delete Mapping Files. When prompted by the message box, “Are you sure you wish to delete these Mapping Files?”, click OK to confirm the deletion. Then you will see that the selected mapping file is removed from the Mapping Files page and the DTD Mapping Files page.
