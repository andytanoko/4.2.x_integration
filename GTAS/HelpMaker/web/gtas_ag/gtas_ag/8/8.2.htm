8.2 Configuring Mapping Files

GridTalk has the capability to provide the mapping engine, which is the process of specifying the way in which data in one structure (e.g. header or content) is transformed into another structure.

A mapping file is a file that is responsible for processing documents. There are available types of mapping files such as:

Mapping File

Purpose

File Extension

XSL

Transforms XML documents.

xsl

Conversion Rule

Converts files from one format to another.

gmr, xml

Reference Doc

References documents for XSL transformation.

xml

DTD

Validates XML documents.

dtd

Schema

Validates XML documents.

xsd

Dictionary File

Validates data in an XML document.

xml

XPath

Extracts data from an XML document.

xml

Java Binary

Converts files from one format to another.

.jar

The mapping file can be created using a specialized mapping rule editor program. When created, it can be inserted into GridTalk.
