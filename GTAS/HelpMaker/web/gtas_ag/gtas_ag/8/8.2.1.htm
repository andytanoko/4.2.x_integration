8.2.1 To create a new XSL mapping file

A XSL file is used to transform a XML file into another XML file of the same of different format. For example, a XSL can be used to transform a proprietary XML document into a RosettaNet XML document. A XSL can also be used to transform a XML file into a text file but this transformation is usually handled by using a Conversion Rule.

Step 1. Click Mapping.

Step 2. Click XSL under Mapping Files. The XSL Mapping Files list view page is displayed where you see a list table of predefined XSL mapping files.

Step 3. In the XSL Mapping Files list view page, click Create Mapping File. The Mapping File detail view page is displayed where you need to enter the fields.

Step 4. Enter Name and then Description.

Step 5. Select XSL for the Type field.

Step 6. Select the file by clicking Browse. The File selector is displayed for you to select the XSL file.

Step 7. Click Save Mapping File. The new mapping file will be displayed in the Mapping Files or XSL Mapping Files list.

Step 8. To add more XSL mapping files, repeat Step 3 through Step 7. If you want to view the mapping file, go to the XSL Mapping Files page where you will see the records listed in the table. Click on the selected mapping file’s hyperlink under Name in the table. If you wish to update the selected mapping file, click Edit Mapping File.

Step 9. To edit the existing or current mapping file, select the mapping file and then click on the corresponding Edit button . The Mapping File page is displayed. Update the entries of the mapping file as necessary. Click Save Mapping File.

Step 10. To delete the existing or current mapping file, tick the Select checkbox of the mapping file you want to delete. When you see a mark appear in that checkbox, click Delete Mapping Files. The message box appears asking you, “Are you sure you wish to delete these Mapping files?” Click OK. The selected mapping file will be removed from the Mapping Files page and XSL Mapping Files page.
