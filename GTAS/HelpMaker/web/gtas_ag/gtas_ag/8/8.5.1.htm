8.5.1 To create a new AS2 Doc Type Mapping

Step 1. Click AS2 Document Type Mapping under the Mapping folder. The AS2 Document Type Mappings list view page is displayed.

Step 2. To create a new AS2 Document type mapping, click Create AS2 Document Type Mapping. The Create AS2 Document Type Mapping detail view page appears.

Step 3. Select Partner. If the partner is not listed, click button to create a new partner.

Step 4. Select AS2 Document Type. If this document type is not listed, click button to create a new AS2 Document Type. For creating Document Type, see Chapter 8.1.1.

Step 5. Select Document Type. If this document type is not listed, click button to create a new Document Type. For creating Document Type, see Chapter 8.1.1.

Step 6. Click Save AS2 Document Type Mapping.

Step 7. To edit the AS2 document type mapping, select the AS2 document type mapping and then click on the corresponding Edit button . The screen appears. Update the entries of the AS2 document type mapping as necessary. Click Save AS2 document type Mapping.

Step 8. To delete the AS2 document type mapping, tick the Select checkbox of the AS2 document type mapping you want to delete. As you see a mark appear in that checkbox, click Delete AS2 Document Type Mappings. The message box, “Are you sure you wish to delete these Mappings?” will appear. To confirm the deletion, click OK. The selected mapping will be removed from the AS2 Document Type Mapping page.
