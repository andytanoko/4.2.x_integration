8.2.3 To create a new reference doc mapping file

A reference doc is a XML file that can be used by other mapping files such as XSL where they can refer to the reference document to extract information from it. When supplied, the XSL mapping rule may use values in the Reference Doc using XPath. An example is a lookup table that provides supplier codes to be filled into an XML document that is being sent out.

Step 1. Click Mapping. Then click Reference Doc under Mapping Files. The Reference Doc Mapping Files list view page is displayed.

Step 2. Click Create Mapping File. The Mapping File detail view page is displayed.

Step 3. In the Mapping File detail view page, enter the mandatory fields.

Step 4. Select Reference Doc for the Type field.

Step 5. Select the file by clicking Browse. The File selector is displayed where you can select a Reference Doc file.

Step 6. Click Save Mapping File. The new mapping file will be displayed in the Mapping Files or Reference Doc Mapping Files list.

Step 7. To add more Reference Doc mapping files, repeat Step 2 through Step 6. If you want to view the mapping file, go to the Reference Doc Mapping Files page where you will see the records listed in the table. Click on the selected mapping file’s hyperlink under Name in the table. If you wish to update the selected mapping file, click Edit Mapping File.

Step 8. To edit the Reference Doc mapping file, select the mapping file and then click on the corresponding Edit button . The Mapping File page is displayed. Update the entries of the mapping file as necessary. Click Save Mapping File.

Step 9. To delete the Reference Doc mapping file, tick the Select checkbox of the mapping file you want to delete. Once a mark appears in that checkbox, click Delete Mapping Files. Then you will be prompted to confirm the message box, “Are you sure you want to delete these Mapping Files?” Then click OK. The selected mapping file will be removed from the Mapping Files page and the Reference Document Mapping Files page.
