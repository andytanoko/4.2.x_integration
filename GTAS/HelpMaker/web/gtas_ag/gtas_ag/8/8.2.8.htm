8.2.8 To create a new XPath mapping file

An XPath mapping file is a file used for extract information from an XML user document in two occasions:

1. The XML document is imported from the backend server into GridTalk but the partner information is not readily known during import, but this information exists in the document itself.

2. The XML document enters a folder in the GridTalk system during partner function execution, and some document information is necessary to be present in the GridDocument header for subsequent processing.

An XPath mapping file can be crafted using an XML editor or any text editor, and then inserted into GridTalk as follows:

Step 1. Click Mapping.

Step 2. Click XPath under Mapping Files. The XPath Mapping Files list view page is displayed.

Step 3. Click Create Mapping File. The Mapping File detail view page is displayed.

Step 4. In the Mapping File detail view page, enter the mandatory fields.

Step 5. Select XPath for the Type field.

Step 6. Select the file by clicking Browse. The File selector is displayed where you can select a XPath file.

Step 7. Click Save Mapping File. The new mapping file will be displayed in the Mapping Files or XPath Mapping Files list.

Step 8. To add more XPath mapping files, repeat Step 3 through Step 7. If you want to view the mapping file, go to the XPath Mapping Files page where you will see the records listed in the table. Click on the selected mapping file’s hyperlink under Name in the table. If you wish to update the selected mapping file, click Edit Mapping File.

Step 9. To edit the XPath mapping file, select the mapping file and then click on the corresponding Edit button. The Mapping File page is displayed. Update the entries of the mapping file as necessary. Click Save Mapping File.

Step 10. To delete the XPath mapping file, tick the Select checkbox of the mapping file you want to delete. As you see a mark appear in that checkbox, click Delete Mapping Files. The message box, “Are you sure you wish to delete these Mapping Files?” will appear. To confirm the deletion, click OK. The selected mapping file will be removed from the Mapping Files page and the XPath Mapping Files page.
