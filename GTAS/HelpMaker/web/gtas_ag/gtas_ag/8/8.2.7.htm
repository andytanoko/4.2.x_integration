8.2.7 To create a new dictionary mapping file

Dictionaries are used to validate the message contents. The dictionary may be imported using the XML/Dictionary configuration view.

The format of the dictionary file is in the form:

<Dictionary>

<elementname >

<value>value1 </value>

<value>value2 </value>

...

<elementname >

</dictionary>

The dictionary contains a list of valid values for each data entity within a message. If the entity appears in the dictionary, its value must match one of the valid values for that entity, otherwise it will fail the dictionary validation.

For example, any element named elementname in the document must content any of the values value1 or value2 . Otherwise, the dictionary check will be deemed to have failed.

Step 1. Click Mapping.

Step 2. Click Dictionary under Mapping Files. The Dictionary Mapping Files list view page is displayed.

Step 3. Click Create Mapping File. The Mapping File detail view page is displayed.

Step 4. In the Mapping File detail view page, enter the mandatory fields.

Step 5. Select Dictionary for the Type field.

Step 6. Select the file by clicking Browse. The File selector is displayed where you can select a Dictionary file.

Step 7. Click Save Mapping File. The new mapping file will be displayed in the Mapping Files or Dictionary Mapping Files list.

Step 8. To add more Dictionary mapping files, repeat Step 3 through Step 7. If you want to view the mapping file, go to the Dictionary Mapping Files page where you will see the records listed in the table. Click on the selected mapping file’s hyperlink under Name in the table. If you wish to update the selected mapping file, click Edit Mapping File.

Step 9. To edit the Dictionary mapping file, select the mapping file and then click on the corresponding Edit button . The Mapping File page is displayed. Update the entries of the mapping file as necessary. Click Save Mapping File.

Step 10. To delete the Dictionary mapping file, tick the Select checkbox of the mapping file you want to delete. As you see a mark appear in that checkbox, click Delete Mapping Files. The message box, “Are you sure you wish to delete these Mapping Files?” will appear. To confirm the deletion, click OK. The selected mapping file will be removed from the Mapping Files page and the Dictionary Mapping Files page.
