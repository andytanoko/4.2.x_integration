9.6.1 To create a new RFC

Step 1. Click Process. The browser will bring you to the list of submenu under Process in the left pane.

Step 2. Click RFC. The RFCs list view page is displayed.

Step 3. To create a new RFC, click Create RFC. The RFC detail view page is displayed.

Step 4. Enter Name.

Step 5. Enter Description.

Step 6. Enter Host/IP Address of the remote machine

Step 7. Enter Port Number for GridTalk to connect to the remote machine

Step 8. Specify Use Command File if you want to trigger a remote command after the files has been exported to the remote machine. If ticked, enter Command File Dir, Command File and, if required, the Command Line. The Command File Dir is the directory on the remote machine where your command program resides. The Command File is the program to trigger to execute the command. The Command Line will contain the parameters used by the command program if any.

Step 9. To save the new RFC, click Save RFC.
