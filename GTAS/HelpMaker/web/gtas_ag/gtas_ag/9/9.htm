9 Process

The Process feature automates the document workflows based on the partner functions configured by a user. It allows you to create new triggers, edit and delete existing triggers that are used to set partner functions. It also associates the type with a partner and specifies the groups, which will contain configured partner functions.

The chapter includes the following:

  • Partner Functions
  • Triggers
  • Process Definition
  • Process Mapping
  • RFC
  • Port
  • Procedure Definition Files (Shell Executable, Java Procedure, Web Service)
  • User Procedures (Shell Executable, Java Procedure, Web Service)
