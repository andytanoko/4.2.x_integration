9.4.1 To view a process instance

The Process Instance tracks a particular instance of a process. When a document tagged as a Process document is sent out, the Process Controller will create an instance for it. This process instance is a read-only view, and show the documents associated with the instance (including acknowledgements or exceptions).

Each process instance is identified with a process instance ID that is unique within the context of either the process originator or the recipient.

The User Tracking ID of the process instance is extracted from the request document according to the User Tracking Identifier, which is configured in the process definition as an XPath referring to a user-selected element of the request document.

For example, if the request document is:

<msg>

<user>John</user>

<document>

<id>123</id>

</document>

</msg>

Then the XPath value can be ";//msg/user";, or ";//msg/document/id";, or any value that represents an element in the request document.

After configuring the ";User Tracking Identifier";, when processes of that process definition are executed, the user tracking ID (eg.”123”) will be extracted from the request document using the XPath value (eg. ";//msg/document/id"; ) specified in the process definition. The ID will be stored in the process instance and displayed alongside the existing process instance ID.

The State of the instance indicates which part of the process it is in. The user can tell if the request has been sent, or an acknowledgement is expected, etc.

The Time Due is the alarm for the process. This alarm will be controlled by the time to acknowledge (if the process instance is expecting an acknowledgement) or the time to perform (if a response document is expected). After this timeout, the system will trigger the corresponding action.

Step 1. Click Process.

Step 2. Click Process Instances. This will bring you to the Process Instances list view page where you can see a list of process instances in the left pane of the page.

Step 3. Select the process instance. Click on the magnifying glass icon next to the process instance you have selected under the Process Instance ID column. The screen appears.

Step 4. If you wish to view documents associated to the process instance you have selected, click the corresponding link under the Folder column.
