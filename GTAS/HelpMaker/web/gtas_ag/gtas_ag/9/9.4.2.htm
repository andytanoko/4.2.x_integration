9.4.2 To cancel a process instance

When a Process Instance is in progress, the user can choose to cancel the process. This option allows the user to stop the process in progress without generating additional retries or exception messages.

If the process instance has failed, the failure reason is shown as well. This can be either a RosettaNet Exception received, or perhaps cancelled by the user.

When a process is cancelled, a 0A1 Failure Notification process will be initiated to the trading partner of the original process that has been cancelled. However, for the failure PIP to be initiated successfully, the appropriate Process Mapping must be configured for 0A1 Failure Notification between you and your trading partner. Refer to Chapter 9.5 for details on configuring Process Mappings.

Step 1. Click Process Instances. This will bring you to the Process Instances panel on the left. When a process is performed, it will appear under the Process Instances folder.

The following information is displayed in the list of process instances:

Process Instance Id

This column indicates the identity of the process instance.

State

This column indicates the state of the process instance. For example,

State

Explanation

Abnormally Completed

Occurs if you have cancelled the process instance as it has failed.

Running

Occurs if the process instance has not been yet completed while still awaiting for acknowledgement or confirmation

Complete

Occurs if the process instance has been successfully completed.

Start Time

Date and time when the process instance started.

End Time

Date and time when the process instance is completed.

Is Failed

This field indicates the status of process instance. True or False. For example, True means that the process instance has failed.

User Tracking ID

Refer to Chapter 9.4.1.

Step 2. To cancel a process instance, tick the checkbox for the process instance, and click Cancel Process Instances. A 0A1 Failure Notification PIP will be initiated for the process instance which is in Running or Complete state.
