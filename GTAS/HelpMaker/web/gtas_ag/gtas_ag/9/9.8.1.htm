9.8.1 To create a new procedure definition file

Step 1. Click Procedure Definition Files. The Procedure Definition Files list view page is displayed:

Step 2. To create a new procedure definition file, click Create Procedure Definition File. The Procedure Definition File detail view page is displayed.

Step 3. Enter Name.

Step 4. Enter Description.

Step 5. Select Procedure Type. This field indicates the type of command to be executed whether it is an executable native program, a Java program, or a Web Service. The available types to choose are Java Procedure, Shell Executable and Web Service.

Step 6. Click Browse to select the Procedure File. This field should be entered as the full path of the executable program, jar file, class file in the local file program, WSDL file or XML file. Click Add/Remove File(s). The file will appear with the blank checkbox under the Procedure File field. If you wish to change to another file, tick its checkbox and then click Add/Remove Files, and repeat this step again.

Step 7. To save the file, save Procedure Definition File. This Procedure Definition File will be used for creating a user procedure.
