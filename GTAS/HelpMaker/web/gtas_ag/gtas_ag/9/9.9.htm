9.9 Configuring User Procedures

The User Procedure is used to allow the user to specify breakpoints to execute external codes/program. The user can configure external codes/program to execute, parameters to pass to external codes/program and expected return value. The user also can configure alert notification actions to perform based on return value.

To configure a user procedure, you just need a three-step process to perform as follows:

Step 1: Configure the external codes/programs to execute

Step 2: Configure the parameters to pass to external codes/program

Step 3: Configure the expected return value and the action to perform based on the return value.
