Default Log Message Templates

The following are the default log message templates provided by GridTalk 4.2:

license.log

Append

yes

Message

[<%System.CURRENT_DATETIME%>] GridTalk License has expired for GridNode <%GridNode.ID%> - <%GridNode.NAME%> on <%License.END_DATE#yyyy-MM-dd%>.
