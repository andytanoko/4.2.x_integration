12.1 Configuring Scheduled Task

The scheduled tasks are configured to run jobs at absolute time, thus if a job is missed for its schedule, for example GridTalk is not running at the time when the job is due to run, the missed job will not be run when the GridTalk resumes operation at a later time.

Step 1. Go to Administration.

Step 2. Click Scheduled Task under the Administration menu. The Scheduled Task list view page is displayed.

Step 3. To create a new scheduled task, click the Create Scheduled Task in the list view page. The Scheduled Task detail view page appears like this by default:

Step 4. Enter the following fields such as:

Task Type:

This field describes the type of task to schedule. The available type is User Procedure which you can select.

Task ID:

Upon selection of Task Type as User Procedure, you are required to enter this field. This field indicates the User Procedure to execute when the scheduled job runs.

Disabled:

This field allows you to specify whether to disable the task temporarily. If checked, no job will be scheduled to run.

Frequency:

This field is the schedule frequency, which you can select Once, Minutely, Hourly, Daily, Weekly or Monthly.

Start Date ( YYYY-MM-DD):

This field is the starting date (in YYYY-MM-DD format) whereby the task configuration takes effect. Depending on the schedule frequency, this might not be the date that the first job is run.

Run Time (HH-MM):

This field is the time (in 24-hour time format) whereby the first job runs. Depending on the schedule frequency, subsequent jobs might not be run at this time.

Times to Run :

This field allows you to specify the number of jobs to schedule for running, following the schedule frequency. Take note that if not specified, jobs will be scheduled for the lifetime of schedule setting until the setting is modified, disabled or deleted.

Step 5. For Start Date, you may choose a date by clicking the icon next to the field. This will bring up the DateTime Picker window showing a calendar like this:

A pink-colored box shows that current date, while a yellow-colored box shows the date in the Start Date text field (if any). To choose a date from another month, select the month from the month drop-down list. To choose a date from another year, click on the red arrow keys beside the year.

To choose a date, use the mouse to click in the box for that date. For example, clicking on box “21 ” will select 2004-05-27. The DateTime Picker window will close upon selecting a date, or clicking on the close window icon (by doing so no date will be chosen).

Step 6. Once you have configured your schedule (refer to Chapters 12.1.1 to 12.1.6), click Save Scheduled Task. The new scheduled task will be added and recorded in the Scheduled Task list view page.
