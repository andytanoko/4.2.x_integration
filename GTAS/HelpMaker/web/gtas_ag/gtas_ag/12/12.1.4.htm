12.1.4 To schedule a minutely jobs

Minutely jobs are scheduled by choosing “Minutely” for the Frequency field.

Additional input required:

Field

Explanation

Every n Minutes

To schedule jobs to run every N number of minutes.

For example, the scheduled task as shown in this figure will run jobs on 24th March 2004 12:00 noon, 24th March 2004 13:30, 24th March 2004 15:00, and so on, for the lifetime of the schedule setting.
