12.1.2 To schedule a daily job

Daily jobs are scheduled by choosing “Daily” for the Frequency field.

Additional input required:

Field

Explanation

Every n Days

To schedule jobs to run every N number of days. Enter 1 to run a job everyday.

For example, the scheduled task as shown in this figure will run jobs on 24th March 2004 12:00 noon, 29th March 2004 12:00 noon, 3rd April 2004 12:00 noon, and so on, for the lifetime of the schedule setting.
