11.4.1 To create a new web service

For own web service configuration, this functionality is used for managing web services that need to be access-controlled.

Step 1. Go to Web Services tab. The Own Services list view page is displayed by default.

Step 2. To create own service, click Create WebService.

Step 3. In the WebService detail view page, key in the following fields:

Service Name

Enter the logical name of the web service you want to create. The logical name should be entered preferably in the non-space delimited string, for example, Service1 . Once done, the name will be referred in the HTTP requests to access the web services. The HTTP request must have the authorization profile for HTTP basic authentication of the partner. The authorization profile should contain the Partner ID (such as the username) and the authentication password assigned to the Partner. Using this HTTP request, you will be able to invoke the web service by providing URL of the following format:

http://<gridtalk_ip:port>/gridtalk/wsrouter/invoke?service=<service_name>

(or https://<gridtalk_ip:port>/gridtalk/wsrouter/invoke?service=<service_name> for SSL connection )

WSDL URL

Enter the URL so that you can get the WSDL file of the web service. For example: http://mywebservice.hello.com/HelloService.asmx?wsdl , where by your partner’s web service client would access the WSDL of your web service “Hello Service ” by providing URL of the following format:

http://<gridtalk_ip:port>/gridtalk/wsrouter/invoke?service= HelloService

(or https://<gridtalk_ip:port>/gridtalk/wsrouter/invoke?service= HelloService for SSL connection )

End Point

Enter the actual Endpoint URL of the web service. For example,

http://mywebservice.hello.com/HelloService.asmx

Step 4. Once you have finished completing the fields, click Save WebService. The new web service will be added to the Own Services list view page. To modify the details of the selected web service, just click on the Edit icon for the record. To delete the selected web service, tick the checkbox against the record and click Delete WebService.
