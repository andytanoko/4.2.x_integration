11.1 Creating a User Procedure to invoke a web service

GridTalk allows you to create Procedure Definition file, which is a WSDL or XML file. Then, you can select that Procedure Definition file to configure a SOAP-enabled User Procedure to invoke a Web Service. SOAP is a protocol that allows messages to be transmitted as XML documents and invokes the capabilities of a Web Service.

In addition, alerts to notify a user of whether the invocation of a Web Service is successful or not are a must to configure before executing a user procedure.
