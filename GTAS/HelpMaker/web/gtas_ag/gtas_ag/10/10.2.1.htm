10.2.1 To export the document into backend system from GridTalk

  • For GridTalk to export the document to the backend listener, you will need to configure the RFC and Port as discussed in the following chapters: To configure an RFC for the backend listener, see Chapter 9.6 Configuring Remote Function Calls (RFC)

  • To configure a Port for the backend listener, see Chapter 9.7 Configuring Ports

  • The default GridTalk Backend Listening Port is 6763. This may be changed globally in this file: <GRIDTALK_HOME>/data/GNapps/conf/default/backend.properties
