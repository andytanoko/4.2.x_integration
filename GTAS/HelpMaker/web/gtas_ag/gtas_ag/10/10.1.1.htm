10.1.1 To import the document into GridTalk

For the backend sender to import the document into GridTalk, you will need to provide the information in sender.ini and/or directory.file as detailed in the following chapters. The backend sender directory is usually located at this location: <GRIDTALK_HOME>/backend/sender

Typically, the sender.jar works in the following fashion:

  1. Pre-process (if required) files picked up from specified directory location(s)

  1. Drop files into a sub-folder outToGT before sending to GridTalk
    1. Files with conflicting names among the directories are renamed

  1. Move files imported to GridTalk into sub-folder outToGT/sentToGT
    1. Files failed to move are transferred to another sub-folder outToGT/fileCollision
    2. Files with conflicting names are renamed
