10.1.3 Directory.file

The directory.file is an XML file that is used when there is a need to pick up files from many different directories. It allows the use of info files for each directory. This is useful for batch import of files for multiple partners and document types.

The following shows a sample of the directory.file:

XML Tag

Usage Description

<Directory>

Root element of the file. Defines one of more Locations to pick up files.

<Location>

Level 2 element. Defines a location to pick up files.

<Name>

Level 3 element. An arbitrary name for the location.

<Directory>

Level 3 element. The physical directory to pick up documents.

<AttachmentDir>

Level 3 element. The physical directory to pick up attachments. Optional.

<InfoFile>

Level 3 element. The information file to apply to all documents picked up in that location.
