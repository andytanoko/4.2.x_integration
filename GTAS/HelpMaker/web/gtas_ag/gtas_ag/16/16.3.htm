16.3 Configuring UCCNet Connections

GridTalk integrates with a program known as GridNode UtAdaptor Extension (or gn-utadaptor-ext) to invoke the UCCNet transport adaptor (UtAdaptor) to send an UCCnet message and to receive the response message synchronously. This program may also integrate with the Inbound Transporter (gt-ibtransporter) program to push the response message to GridTalk’s Document Inbound.

(Take note that t his program is only available when you purchase this feature. There is documentation for Inbound Transporter that comes with the GridTalk 4.2 software.)
