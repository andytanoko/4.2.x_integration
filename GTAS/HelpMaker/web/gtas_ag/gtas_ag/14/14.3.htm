14.3 Viewing Archived Document Detail s

The Archived Document detail page is very similar to the normal Document detail page. It consists of six tab pages of information:

General

This tab contains the general document information, such as the attributes of the user document and the process instance id of the process instance the document is associated with. Refer to Chapter 6.1.1 of GridTalk 4.2 User Guide for details on the fields shown.

Traceability

This tab contains the information to trace the source and flow of the document. It contains the following fields that are not contained in the normal Document detail page:

Audit File

The original raw data received/sent from/to partner that constitutes the user document. This can be downloaded and saved on user system by clicking on the hyperlink.

Receipt Audit File

The original raw data received/sent from/to partner for acknowledgment of receipt. This value is only present currently for AS2 documents. This can be downloaded and saved on user system by clicking on the hyperlink.

Transaction Status

The status of the document transaction. This field is mainly used for AS2 documents. This field would also capture problems with sending for all types of documents.

Refer to Chapter 6.1.2 of GridTalk 4.2 User Guide for details on the other fields shown.

Partner

This tab shows the sender and receiver information of the document. The certificates used by the send and the receiver to transact the document, if used, are shown, and clicking on the corresponding hyperlink would display the details of the certificate. Refer to Chapter 6.1.3 of GridTalk 4.2 User Guide for details on the other fields shown.

Communication

This tab shows the timestamps and channel information of the transaction. Refer to Chapter 6.1.4 of GridTalk 4.2 User Guide for details on the fields shown.

Attachments

This tab shows the attachments that are included with the user document. It is only enabled for selection if there are attachments. Refer to Chapter 6.1.5 of GridTalk 4.2 User Guide for details on the fields shown in this tab.

Error

This tab shows the errors encountered during the archival of the document. It is only enabled for selection if there are errors.
