6.3.3 To create a new JMS message template

The JMS message template contains the following fields you will want to modify to create a new Message Template:

Field

Explanation

Name

The name of the Message Template

JMS Destination

The target JMS Destination to send the alert messages to

Message Properties

Additional JMS message properties to set into the JMS alert message

Message

The message content of the alert message

Let us create the simple scenario where you can configure alert message to send as JMS message to notify your JMS-enabled backend systems on occasions that a process instance has failed.

Step 1. To create a new message template, click Create Message Template. Select the Message Type for the template. You are given the options to determine if the message type in the template is Email, Log, or JMS. Select JMS. The screen appears.

Step 2. Enter Name. For example, Process Instance Failure

Step 3. Specify the JMS Destination. Select the target JMS Destination from the listbox display the list of JMS Destinations that you have created earlier. If the target JMS Destination has not been created, click to create a new JMS Destination.

Step 4. Specify message properties for the JMS message, if necessary. Click Add Property to add a new message property. Enter the key, data type, and value for the message property. OFRT is supported in the Value field. 







To remove a property, tick the checkbox against the property row, and click “Remove Property”.

Step 5. You may compose any kind of text message in the Message textbox – this depends on the interface agreement with the JMS-enabled backend system. For example, the message can present in the form of key-value pairs, or user-friendly email-like message.

Step 6. To save the new message template, click Save Message Template. The new message template will be displayed in the list table of the Message Templates page.

Step 7. To edit the existing or current message template, select the message template and then click on the Edit button . Update the entries for the Message Template as necessary. Next click Save Message Template.

Step 8. To delete the existing or current message template, tick the Select check box of the message template you want to delete. Click Delete Message Template. The selected message template will be removed from the list.
