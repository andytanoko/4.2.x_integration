6.6.2 Alert Triggers and Criteria Evaluation

Alert Triggers exist at 5 levels, from Level 0 to Level 4, with each level having its own defined set of criteria for an Alert Trigger at that level to be in effect. Criteria become more stringent as it goes up a level. Level 1-4 Alert Triggers are applicable only to Alert Types in the Document Event, Processing Errors, and User Specific areas.

Criteria evaluation is done from the highest level (Level 4) to the lowest level (Level 0). Generally, if a match is not found at a level, the next lower level criteria is evaluated and if a match is found then that Alert Trigger will be in effect. Otherwise, the next lower level criteria are again evaluated until a match is found or until the lowest level is reached. If at Level 0, there is an Alert Trigger configured for the relevant Alert Type, then the Level 0 Alert Trigger will be in effect.
