6.5.2 The Email Code Reference File

The Email Code Reference File is used for configuring email codes for use in email alert notifications. It refers to the emailcode -ref.xml , which you need to manually add in the email codes for the following:

  • Alert Trigger configuration (using Email code and/or Email code xpath);
  • Response Track Record configuration (using email code xpath); and
  • Message Template configuration (using RecipientList EMAIL_CODE or EMAIL_CODE_XPATH)

This file is in Xml format that defines a collection of EmailCodes. Each EmailCode may map to:

· 0-n number ofEmail addresses

· 0-n number of Roles

· 0-n number of Users
