6.3.1 To create a new email message template

The email message template contains the following fields you will want to modify to create a new Message Template:

Field

Explanation

Name

The name of the Message Template

Description

A short description of the Message Template

From

The sender email of the email notification

To

The recipient(s) email of the email notification

Cc

The Cc recipient(s) email of the email notification

Subject

The subject of the email notification

Message

The message content of the email notification. This content can be a mixture of text and Object Field Reference Tags (eg. Document Type : <%GridDocument.U_DOC_DOC_TYPE%>

Let us create the simple scenario where you can configure alert message to notify the user of the GridTalk license expiry.

Step 1. To create a new message template, click Create Message Template. Select the Message Type for the template. You are given the options to determine if the message type in the template is Email, Log, or JMS. Select Email. The screen appears.

Step 2. Specify the Content Type. The only content type supported currently is Text, select it.

Step 3. Enter Name. For example, GridTalk License Expiry

Step 4. To enter the From field, go to the Object to select User . You will find that the User object contains the available fields such as “ admin ”, “guest” and the names of user accounts that were earlier created. Select “ Admin ” for the field. To insert the corresponding Object field reference tag, <#USER=admin#> in the From field, place the cursor on the From field and click Insert.

Step 5. To enter the To field, go to the Object to select User. Select “ guest ” for the field. To insert the corresponding Object field reference tag, <#USER=guest#> in the To field, place the cursor on the To field and click Insert.

Step 6. Repeat Step 5 for the CC field, if each message needs to be copied to other recipients.

Step 7. To enter the Subject field, go to the Object to select License. For the License object, see Appendix. Next select “Product_Version” for the field. To insert the corresponding Object field reference tag, <%License.PRODUCT_VERSION%> in the Subject field, place the cursor on the Subject field and click Insert.

Step 8. To enter the Message textbox, you may wish to compose the message like “ Please be informed that your license is going to expire on <%License.END_DATE%> “. To do this, just type in the textbox. To insert the tag in the message, go to the Object to select License. Select “END_DATE” for the field. Place the cursor on the textbox and click Insert.

Step 9. To save the new message template, click Save Message Template. The new message template will be displayed in the list table of the Message Templates page.

Step 10. To edit the existing or current message template, select the message template and then click on the Edit button . Update the entries for the Message Template as necessary. Next click Save Message Template.

Step 11. To delete the existing or current message template, tick the Select check box of the message template you want to delete. Click Delete Message Template. The selected message template will be removed from the list.
