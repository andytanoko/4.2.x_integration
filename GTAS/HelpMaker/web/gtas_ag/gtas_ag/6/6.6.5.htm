6.6.5 To trigger an alert level ranked 3

To create a new alert trigger level 3:

Step 1. Click Alert. From Alert, click Alert Trigger. Then click Level 3. The Level 3 Alert trigger list view page with the following columns in the table is displayed.

Edit

This field allows you to update the properties of the selected alert trigger.

Document Type

This field indicates the document type criterion to match.

Partner Type

This field indicates the partner type criterion to match.

Partner Group

This field indicates the partner group criterion to match.

Alert Type

This field indicates the type of alert that the trigger is defined for.

Alert

This field indicates the Alert to be raised when the trigger takes effect.

Step 2. Click Create Alert Trigger.

Step 3. Repeat Step 3 through Step 4 as discussed in the Chapter 6.6.4 for creating alert trigger level 2.

Step 4. Select Partner Group to match. If the partner group is not found in the list, click on the icon to create a new partner group.

Step 5. Once done, go to Step 5 in the Chapter 6.6.4 for creating alert trigger level 2.
