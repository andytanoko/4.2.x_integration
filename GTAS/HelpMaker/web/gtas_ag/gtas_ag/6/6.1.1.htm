6.1.1 To create a new alert

Step 1. Click Administration tab. This browser will take you to the Administration navigation panel. Click Alert in the navigation panel. The Alerts list view page is displayed. You will expect to see the list of Alerts earlier created in the Alerts list view page.

Step 2. To create a new alert, click Create Alert. The Alert detail view page appears.

Step 3. Enter Name and Description.

Step 4. Select Type.

Step 5. Go to Actions. Select the action from the left listbox to be added to the right listbox. Click on the forward arrow button to add the action. Click on the reverse arrow button to remove the action from the listbox. Use the up and down arrows to organize the actions. But if the action you want to select is not listed in the left listbox, just click on the icon to create an action. An alert is defined based on the actions. If no action is selected, it is not possible to create an alert.

Step 6. To save the new alert, click Save Alert. The new alert will be displayed in the list table of the Alerts page.

Step 7. To edit the existing or current alert, select the alert and then click on the Edit button . Update the entries for the alert as necessary. Then click Save Alert to save the changes.

Step 8. To delete the existing or current alert, tick the Select check box of the alert you want to delete. Click Delete Alert. The selected alert will be removed from the list
