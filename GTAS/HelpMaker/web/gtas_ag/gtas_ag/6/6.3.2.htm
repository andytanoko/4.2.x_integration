6.3.2 To create a new log message template

The log message template contains the following fields you will want to modify to create a new Message Template:

Field

Explanation

Name

The name of the Message Template

Location

The file to log the alert message to (relative to data/alert/log directory)

Append

Whether to append the alert message to the log file or overwrite

Message

The message content of the alert message

Let us create the simple scenario where you can configure alert message to log on occasions that document mapping has failed.

Step 1. To create a new message template, click Create Message Template. Select the Message Type for the template. You are given the options to determine if the message type in the template is Email, Log, or JMS. Select Log. The screen appears.

Step 2. Enter Name. For example, Document Mapping Error

Step 3. Enter Location. For example, mapping_errors.log

Step 4. To append each alert message to the log file, tick the Append checkbox.

Step 5. To enter the Message textbox, you may wish to compose the message like:

[<%System.CURRENT_DATETIME#yyyy-MM-dd HH:mm:ss.SSS%>] Mapping Rule <%Mapping.NAME%> has failed for document <%GridDocument.U_DOC_FILENAME%>, Message: <%Exception.MESSAGE%>.

To do this, just type in the Message textbox. To insert the System.CURRENT_DATETIME tag in the message, go to the Object and select System. Select “CURRENT_DATETIME” for the field. Place the cursor on the textbox and click Insert. You may have to enter the date format yourself before the closing tag. Likewise for Mapping.NAME and other tags, the text before the “.” is the Object, and the text on the right of “.” will be the Field.

Step 6. To save the new message template, click Save Message Template. The new message template will be displayed in the list table of the Message Templates page.

Step 7. To edit the existing or current message template, select the message template and then click on the Edit button . Update the entries for the Message Template as necessary. Next click Save Message Template.

Step 8. To delete the existing or current message template, tick the Select check box of the message template you want to delete. Click Delete Message Template. The selected message template will be removed from the list.
