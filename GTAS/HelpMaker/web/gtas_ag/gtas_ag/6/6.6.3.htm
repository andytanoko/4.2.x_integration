6.6.3 To trigger an alert level ranked 1

To create a new alert trigger level 1:

Step 1. Click Alert. Click Alert Trigger to select Level 1. The Level 1 Alert trigger list view page with the following columns in the table is displayed.

Edit

This field allows you to update the properties of the selected alert trigger.

Document Type

This field indicates the document type criterion to match.

Alert Type

This field indicates the type of alert that the trigger is defined for.

Alert

This field indicates the Alert to be raised when the trigger takes effect.

Step 2. Click Create Alert Trigger. Then the Alert Trigger 1 detail view page is displayed.

Step 3. Select Document Type to match. If the document type is not listed, just click on the icon to create a new document type.

Step 4. Select Alert Type.

Step 5. Once the alert type is selected, select Alert. If the alert you want is not listed, click on the icon to create a new alert.

Step 6. Next proceed to the Recipients section. Select Type. For that Type, select the corresponding Value as explained in the table below.

Type field

What to do with the Value field

User

Select User. If the user is not listed, just click on Create User to create a new user account.

Role

Select Role

Email Address

Enter the user’s email address

Email Address XPath

Enter XPath for extracting email address from the document

Email Code

Enter Emailcode (configured in the email code reference file)

Email Code XPath

Enter XPath for extracting email code from the document. The extracted email code will be used for lookup in the email code reference file

Later add the recipients of the alert message. This is in addition of the fixed recipients defined in an email Message Template. The Message Template must contain a <%RecipientList.DYNAMIC_RECIPIENTS%> tag in the To field.

Step 7. If you want the alert trigger to be disabled, un-tick the Enabled checkbox. Take note that if the trigger is in effect but it is not enabled, the alert will not be raised.

Step 8. If you want to attach document in email notifications, tick its checkbox. Take note that if the alert raised results in email notifications, this will apply only to alert types in document event, processing errors and user specific areas.

Step 9. Once done, click Save Alert Trigger. The new alert trigger will be added to the list.

Step 10. To edit the existing or current alert trigger, click on the Edit button next to the alert trigger you have selected. The browser will show you the properties of the selected alert trigger. To update the entries of the trigger as necessary, click Edit Alert Trigger. Once the changes have been made, click Save Alert Trigger.

Step 11. To delete the existing or current alert trigger, select the check box of the alert trigger you want to delete. Click Delete Alert Triggers. The selected trigger will be removed from the Level 1 Alert Triggers list and the All Alert Triggers list.
