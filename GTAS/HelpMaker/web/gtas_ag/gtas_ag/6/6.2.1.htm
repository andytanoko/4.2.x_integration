6.2.1 To create a new action

Step 1. Click Action. The Actions list view page is displayed. You will expect to see the list of actions earlier created in the Actions table.

Step 2. To create a new action, click Create Action. The Action detail view page appears.

Step 3. Enter Name and Description.

Step 4. Select Message Template. If the message template is not listed, click on the icon next to the Message Template field to create a message template.

Step 5. To save the new action, click Save Action. The new action will be displayed in the list table of the Actions page.

Step 6. To edit the existing or current action, select the action and then click on the Edit button . Update the entries for the action as necessary. Click Save Action.

Step 7. To delete the existing or current action, tick the Select check box of the action you want to delete. Click Delete Action. The selected action will be removed from the list.
