13.4 Scheduling housekeeping job s

Step 1. Click the Administration tab. This will bring you to the Administration page.

Step 2. Click Housekeeping Settings. Then the Housekeeping Settings list view page appears like this:

Step 3. Enter the following fields for the Housekeeping setting:

Temporary Files

This field requires you to specify the maximum age (in days) of the temporary files to keep. Temporary files older than the value will be purged from the file system.

Log Files

This field requires you to specify the maximum age (in days) of the log files to keep. Log files older than the specified value will be purged from the file system.

Ended PF Runtime Records

This field requires you to specify the maximum age (in days) of the ended partner function runtime records to keep. Runtime records ended days more than the specified value will be purged from the file system.

Step 4. To confirm the setting, click Apply.

Step 5. Next, click Scheduled Task under the Administration menu. This will display you the Scheduled Task list view page.

Step 6. From the list view page, you will find the “Housekeeping Info” scheduled task. Click the Edit button to bring up the detail view page for the housekeep scheduled task.

Step 7. Uncheck the Disabled checkbox to enable the housekeeping schedule.

Step 8. To schedule the job regularly, change the Frequency and timing as detailed in Chapter 0.
