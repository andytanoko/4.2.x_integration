6.1.4 Communication Information

Clicking on the Communication tab pane displays the data in the Communication, Channel, Routing and Port sections.

The Communication section contains the date and time details when the documents have been exchanged between trading partners.

Field

Description

Date/Time Imported

Date and time when the document was imported into GridTalk

Date/Time Sent

Date and time when the document was sent to the receiver

Date/Time Transacted

Date and time when the transaction was completed

Date/Time Received

Date and time when the document was received from the sender

Date/Time Exported

Date and time when the document was exported out of GridTalk

Date/Time Viewed

Date and time when the document was viewed by the user

The Channel section contains information about the sender’s or receiver’s channel when exchanging documents.

Field

Description

Channel Name

Name of the channel used for sending/receiving document messages

Encryption Level

Level of encryption set to ensure the security of the document messages received from/send to partners.


The Routing section contains the information which indicates whether the document is directly routed from / to partner or routed via GridMaster.

Field

Description

Route via

If the document is directly routed to partner, the field will show DIRECT. It tells you that the document is sent to the partner when the partner is online.

The Port section contains information about the port used to export the business documents out of GridTalk.

Field

Description

Export via

Type of port to send documents to the desired destination or to the backend system such as SAP
