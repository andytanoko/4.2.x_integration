6.3.2 To manually send a document

Manual send operation can only be performed on documents in the Import folder, also provided that a manual send partner function processing is already defined by your Administrator for that type of document that you select for manual send. This function may be used to resend your document if it fails to send out to your partner possibly due to configuration errors, or used as part of your business process flow to allow you to make modification to a received document and send it back to the original sender partner.

Step 1. In the Import Documents list view page, select the document files to be sent.

Step 2. Click Send. You will be prompt to confirm sending:

Step 3. Click Ok to send the selected documents.
