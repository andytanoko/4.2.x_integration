6.1.1 General Information

Clicking on the General tab pane displays the data in the General and Embedded sections.

The General section contains the general information kept within GridDoc. It is used for tracking and audit purposes. Most of the information is generated by the system. The fields are explained in the table below:

Field

Description

GridDoc. ID

Unique ID number in each Document Folder used for tracking purposes

GridDoc. Filename

Filename of the GridDoc

Folder

Document folder (if any) of the business document

Process Instance

Process Instance ID of the process the business document participates in (applicable for Inbound, Outbound and Export documents).

User Tracking ID

User tracking identifier extracted from the business document if any.

Transaction Status

Transaction status of the business document. For example, it will show the processing status of an Inbound or Outbound AS2 document.

The Embedded section contains information about the business document.

Field

Description

Doc. No

Document number (if any) of the business document

Doc. Filename

Filename of the business document

Doc. Version

Version (if any) of the business document

Doc . type

Document type (as defined by user) of the business document

Doc. File type

File type of the business document, e.g., CSV file

Doc. File Size (kb)

File size of the business document

Receipt Audit File

Filename of the received audit document. For example, it will capture the filename of the returned AS2 MDN from the partner.

Audit File

Filename of the audit document.
