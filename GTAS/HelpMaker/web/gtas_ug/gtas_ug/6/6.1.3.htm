6.1.3 Partner Information

Clicking on the Partner tab pane displays the data in the Sender and Receiver sections.

The Sender section contains the information about the sender of the document. Note that this information will only be present after the document has been received in the Inbound folder and will primarily be displayed for the receiver, rather than the sender.

Field

Description

Sender GridNode. ID

GridNode ID of the sender partner if any.

Sender Business Entity ID

Business Entity ID of the sender derived from the sender’s User ID

Sender Partner ID

Sender’s partner ID as defined by the receiver

Sender Partner Name

Sender’s partner name according to the receiver’s definition

Sender Partner Type

Sender’s partner type defined by the receiver

Sender Partner Group

Sender’s partner group according to the receiver’s definition

Sender User ID

User ID of the sender

Sender User Name

User Name of the sender

The Receiver section contains the information about the receiver of the document. Note that this information will only be present after the document has been sent in the Outbound folder. It will primarily be displayed for the sender, rather than the receiver.

Field

Description

Recipient GridNode. ID

GridNode ID of the receiver partner if any

Recipient Business Entity ID

Business Entity ID of the receiver

Recipient Partner ID

Receiver’s partner ID as defined by the sender

Recipient Partner Name

Receiver’s partner name according to the sender’s definition

Recipient Partner Type

Receiver’s partner type defined by the sender

Recipient Partner Group

Receiver’s partner group according to the sender’s definition
