6.5.2 To manually export a document

This manual export may be used if you fail to export the documents into the backend system automatically when the document is received or imported. Using this feature, you can manually export documents from the Import/Inbound folders to the Export folder.

Step 1. In the Import Documents or Inbound Documents list view page, select the document files to be exported.

Step 2. Click on the Export link on top of the list view. When the message, “Export Specified Documents?” is prompted, click OK.

Step 3. The default partner function for Manual Export will be used if you do not define your own Manual Export partner function. See more details in the GridTalk 4.2 Administration Guide for creating a partner function. If the manual export partner function configures Exit to port activity, the selected documents are now exported according to the specified port.
