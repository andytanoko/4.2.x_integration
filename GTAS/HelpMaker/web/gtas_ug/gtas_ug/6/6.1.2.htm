6.1.2 Traceability Information

Clicking on the Traceability tab pane displays the data in the Traceability and Partner Function sections.

The Traceability section contains the information used by GridTalk for synchronization and reconciliation of documents during the workflow process. This information is useful when you receive back a document that has been created based on a source document that you sent out, and you need to refer back to that source document to process it.

Field

Description

Sender GridDoc. ID

GridDoc ID of the business document at the sender

Recipient GridDoc. ID

GridDoc ID of the business document at the recipient (if known)

Ref. GridDoc. ID

GridDoc ID of the reference GridDoc document ( if any) (the source GridDoc)

Ref. Doc. No

Document Number of the reference business document

Ref. Doc. Filename

Filename of the business document that is the source of the reference document

Source Folder

Indicates where the business document flows from Import Folder of the reference document

The Partner Function section contains information about the partner function that is triggered to handle the document.

Field

Description

Partner Function ID

Partner Function ID of the partner function that handles the business document.
