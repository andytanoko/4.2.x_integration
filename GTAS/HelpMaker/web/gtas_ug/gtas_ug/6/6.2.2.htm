6.2.2 To import a document

Before importing documents into GridTalk, make sure that a partner must be first created.

Step 1. Click “Import” in the navigation tree (which is accessed by way of the “Documents” tab). The browser will bring you to the Import Documents folder.

Step 2. In the Import Documents page, click Import.


Step 3. Select the business entity for the Sender ID field. If the business entity you want to select is not found, click on the icon to create a new business entity.

Step 4. Select Document Type. If the document type you want to select is not found, click on the icon to create a document type.

Step 5. To select the file from your system, click Browse. The Choose File window appears. Once the file is selected and displayed at the Files field, the Add/Remove File(s) will be enabled. Click this hyperlink. You will see the selected file, together with its blank checkbox, appears. You may add more files if you need to import more files. If you tick the checkbox, and click Add/Remove Files, the selected file will be removed.

Step 6. Next go to Recipients where you can select which partners are to receive your files. If the partner you want is not listed in the left listbox, click on the icon to create a new partner.

Step 7. If you wish to attach files from your system, click Browse. The Choose File window appears. Once the file is selected and displayed at the Attachments field, the Add/Remove File(s) will be enabled. Click this hyperlink. You will see the selected file, together with its blank checkbox, appears. You may add more files if you need to import more files. If you tick the checkbox, and click Add/Remove Files, the selected file will be removed.

Step 8. Next click the Import hyperlink to send the files to the selected partner. You will see the files you have selected are imported into GridTalk and displayed in the All Documents and Import Documents list view pages.

Step 9. To select and delete the files from the Import folder, go to the left panel of System Folders and click Import. Select the document and tick its checkbox. Then click Delete Documents. You will be prompted to the Microsoft Internet Explorer window asking you, “Are you sure you wish to delete these Documents?” If you want to delete them, click Yes. You will see the deleted file is removed from the table at the Import Documents list view page.
