6.2.1 To view the import folder

Click Import under the Documents folder in the navigation pane. This will bring you to the Import Documents folder where you can import, send and delete business documents in the import folder.

The columns in the table represent the following information:

GridDoc ID:

It is the unique ID number in the Import Folder used for tracking purposes.

Has Attachments:

Indicates whether the business document has any files attached. If Yes, it means the business document has attached files. If No, the business document has no attached files.

Doc. Type:

Type of the business document. For example, 3A4 Purchase Order Request, Unclassified, etc.

Doc. No:

Document number (if applicable) of the business document.

Doc. Filename:

Filename of the business document. For example, PO.xml.

Recipient Partner ID:

Intended recipient’s Partner ID as defined by the sender.

Date/Time Imported:

When the document was imported into GridTalk.

User Tracking ID:

Not applicable for Import Folder.

Process Instance UID:

Not applicable for Import Folder.
