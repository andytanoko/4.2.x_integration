1 About the Guide

This GridTalk 4.2 User guide is especially designed for the end users. The guide contains the step-by-step instructions and visual screens to enable you to perform the simple tasks while you are getting acquainted with the features in GridTalk 4.2.

It covers mainly the following features which you should be able to perform the tasks:

· Registry

Instructions on how to create business entities to identify and engage partners in the business process.

· Partner

Instructions on how to create partner types and groups to describe an active partner when you are engaged in the document transaction with your partner.

· Document

A quick summary on the different movements of documents, such as Import, Export, Outbound and Inbound. Information on how the active documents are exported, imported, transacted, time-stamped, received and sent to trading partners.

· Document Search

A quick summary of how the active and existing documents are searched for based on the search criteria you set up.
