3.2 Links

While using GridTalk , you will notice that some links are used on different pages to indicate common operations. A link is an icon or section of text that you may click on to navigate or invoke some kind of action. In GridTalk, these links are usually underlined to make it easy to identify them on the screen.

Here follows a list of the most commonly used links in GridTalk.

Link

Comment

Admin

Refers to the admin area of the application. When in the admin area, it will display “User” and clicking it will take you back to the user area.

Logout

The Logout link ends your GridTalk session and returns you to the Login page.

Refresh

This link is used to refresh the list view page with the most current information. For example, if another user edits one of the records, its details will not be updated until you refresh the list view or go to a different page and return.

Delete

This link is used to delete a record from the list view page. For more details, see Chapter 3.4 List View Page.

Create

This link is used to create a record to be added in the list view page. For more details, see Chapter 3.4 List View Page.

Save

This link is used to save the changes that you have made to the record. For more details, see Chapter 3.5 Detail View Page.

Cancel

This link is used to exit the Detail View Page without editing anything to a record. For more details, see Chapter 3.5 Detail View Page.

This link is used to create an item in the Detail View Page.
