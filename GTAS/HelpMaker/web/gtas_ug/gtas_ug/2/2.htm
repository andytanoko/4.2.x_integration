2 Getting Started with GridTalk

To launch GridTalk, use the web browser by keying in the URL field. Then you will see the GridTalk Login appearing in the browser.

Before starting up the GridTalk, ensure that you have the GridTalk 4.2 Installation Guide to follow the sequences.

Step 1. Next launch the web browser and key in your URL in the site address. Take note that this URL will be provided to you by your administrator. A few moments later, the browser will establish a web page where you see the GridTalk Login as shown in the figure below.

Step 2. Enter the user name.

Step 3. Enter the password.

Step 4. Select the Timezone. For information on the Timezone, refer to GridTalk 4.2 Administrator Guide.

Step 5. Click on the Login button. Then the browser will show the Registered page if you have already registered for service to GridTalk. After initial registration, you will normally see the Documents tab that displays the Import Document page by default during the next login. To explore how to use GridTalk, see Chapter 3 Using GridTalk User Interface.
