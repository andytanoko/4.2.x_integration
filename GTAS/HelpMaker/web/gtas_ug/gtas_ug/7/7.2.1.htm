7.2.1 To run a search query and view searched documents

Step 1. To start searching, click on the name of the Search Query on the left side of the browser in the navigation tree view. This will bring you to the Searched Documents list view page where you see all documents that have satisfied the criteria you have set and saved earlier.

Step 2. To view the details of a document from the Searched Documents list view page, click the hyperlink under the Folder column of the document to view. To view the business document, click on the hyperlink under the Doc. Filename column.

Step 3. If you want to edit the search query, go to the Search Query list view page and click on the Edit icon next to the selected Search Query under Search Query in the list view.

Step 4. Make the necessary additions, removals or modifications of conditions and save the search query.

Step 5. To view the results of applying the updated Search Query, repeat Step 1.
