7.1.1 To create a search query for searching by Grid Document

Step 1. Click “Search” in the navigation tree (which is accessed by way of the “Documents” tab). The browser will bring you to the Search Query list view page.

Step 2. In the Search Query list view page, click Create Search Query. Then the Create Search Query detail view page will show up for you to create a new search query.

Step 3. Enter Name and Description for that search query you are creating.

Step 4. Next you will have to set the criteria for the Search Query. There are two types of criteria – Grid Document and User Document.

Step 5. If you choose Grid Document, the Field / XPath list will be displayed for you to select the GridDocument field.

Step 6. Next choose the operator you want for this criteria.

Step 7. Next enter the value you want to search for.

Step 8. To add another search criterion, click on the Add Condition link. Then you will see the second new row appear below the completed first row. To remove a condition, tick the checkbox next to the selected condition, and then click on the Remove Condition link. The condition(s) checked will be deleted.

Step 9. To share the search query with the other user in the application, check the checkbox Public. By default, the search query will only be viewed by the owner who created this query.

If the search query is set as Public, the other user is allowed to modify the query but only owner or the user with Admin role is allowed to delete such query.

Step 10. To save the Search Query, click on the Save Search Query link. The search query’s name will be reflected and shown under Search in the navigation tree. (See Step 7 in Chapter 7.1.2).

Step 11. Refer to Chapter 7.2 to run the search query and view the search results.
