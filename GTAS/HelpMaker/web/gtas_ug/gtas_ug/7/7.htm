7 Document Searching

This feature is designed to allow the user to search the documents in the four system folders. You can search for documents either by the fields in the GridDocument or by using XPaths if the user document is in XML format. This is done by setting and saving the criteria in the form of a search query.

You may make as many queries as you require.

The criteria for the queries are specified by means of defining filters based on the contents from GridDocument (required) and User Document (optional).
